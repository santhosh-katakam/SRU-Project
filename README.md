# RescueRide

A responsive web application for finding nearby mechanics and towing services instantly. Built with HTML, CSS, JavaScript, and Leaflet.js for mapping functionality.

## Features

- Real-time GPS location tracking
- Interactive map with nearby mechanics
- Service details and ratings
- UPI payment integration with QR code
- Navigation and direct calling
- Mobile-responsive design

## Technology Stack

- HTML5
- CSS3
- JavaScript
- Leaflet.js for maps
- OpenStreetMap tiles
- UPI payment integration

## Getting Started

1. Clone this repository
2. Open `index.html` in a web browser
   - For geolocation features, serve via HTTPS or localhost
   - VS Code Live Server extension recommended

## Services

- Towing
- Flat Tire
- Fuel Delivery
- Battery Jump
- Lockout Services
- Minor Repairs
